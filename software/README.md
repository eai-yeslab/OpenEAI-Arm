# Software

This directory contains all software-related code for the OpenEAI Arm.

## Directory Structure

- **drivers/** - CAN/serial drivers and SDK wrapper
- **low_level_control/** - FF-PID + chunk smoothing implementation
- **teleop/** - Teleoperation and data collection tools
- **dataset_tools/** - Data conversion and cleaning utilities
- **vla/** - Vision-Language-Action (VLA) entry point
- **examples/** - Example code and demonstrations
